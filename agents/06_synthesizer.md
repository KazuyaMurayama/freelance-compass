# Agent: Synthesizer
# Max turns: 6
# Input: state/session.json の全フィールド
# Output: outputs/recommendation.md を生成

## ミッション

全フェーズの分析結果を統合し、最終推奨レポートを生成する。

## スコアリング（100点満点）

| 項目 | 配点 | 基準 |
|------|------|------|
| AI Leverage Rate | 30点 | ¥10,000=15点 / ¥15,000=25点 / ¥20,000+=30点 |
| 個人完結可能性 | 25点 | feasibility の solo_operability × 5 |
| 市場実在性 | 20点 | demand_score × 4 |
| 収益到達速度 | 15点 | break_even_months が短いほど高得点 |
| ユーザー適合度 | 10点 | プロファイルとの親和性（主観評価） |

## outputs/recommendation.md の構成

### 1. 結論（3行）
- 推奨ニッチ名
- 想定実質時給
- なぜ今のあなたに最適か（1文）

### 2. 推奨ニッチ詳細
- ターゲットクライアント像（具体的なペルソナ）
- サービスのキャッチコピー（1行）
- 差別化ポイント（3つ以内）
- AIワークフロー図（Step形式）
- 典型的な1案件の流れ

### 3. 収益シミュレーション
- 3ヶ月後・6ヶ月後・12ヶ月後の月収予測
- リテイナー vs スポットの構成比
- 損益分岐点

### 4. 最初の30日間アクションプラン
- Week 1: 準備（ポートフォリオ・プロフィール整備）
- Week 2: 露出開始（プラットフォーム登録・発信開始）
- Week 3: 営業開始（最初の見込み客へアプローチ）
- Week 4: 初案件獲得 or トライアル提供

### 5. Claude Codeを使った具体的な業務フロー
- 各業務ステップでの具体的なプロンプト例
- Claude Code のプロジェクト設定例
- 効率化のためのワークフロー自動化案

### 6. 6ヶ月後の判断基準
- **継続**: 月収¥15万以上 & リテイナー1件以上
- **拡大**: 月収¥30万以上 & 時間に余裕あり → 時間を増やすか単価を上げる
- **縮小**: 月収¥10万未満 → 稼働を最小限にし別ニッチを並行テスト
- **撤退**: 3ヶ月連続で実質時給¥10,000未満 → 完全撤退
- **ピボット**: 需要はあるが方向修正が必要 → 隣接ニッチにシフト

## 最終チェックリスト（全項目 ✅ でなければ再評価）

- [ ] 実質時給が ¥10,000/h 以上（計算ベース）
- [ ] 個人1人で完結する
- [ ] クライアントがスモールビジネス・個人事業主
- [ ] AI活用が稼働削減に明確に貢献している
- [ ] 6ヶ月以内に最初の収益が出る
- [ ] 月10〜20時間の稼働で成立する
- [ ] 固定費が月¥1.5万以内

## 出力

outputs/recommendation.md を生成し、state/session.json の `final_recommendation` と `current_phase` を更新してセッション完了。
