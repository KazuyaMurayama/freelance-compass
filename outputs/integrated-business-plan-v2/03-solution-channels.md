## 3. ソリューション設計とMVP

### 3.1 3層サービスの詳細

**L1: ¥60,000/月（3ヶ月固定PoC）**

| 項目 | 内容 |
|------|------|
| ML予測 | 1本（**離脱予測 or 広告ROI予測** を初回推奨。需要予測は中堅向け） |
| MTG | 月1回30分レビューMTG（人） |
| 補助金 | IT導入補助金併用前提（実質負担 **¥30,000/月**） |
| 設計思想 | 「効いたらL2」の階段。3ヶ月で効果実証 → L2移行 |

> **L1初回提案は「離脱予測」または「広告ROI予測」を推奨。** 需要予測・在庫は年商10億超の中堅向けに限定。

---

**L2: ¥170,000/月（人+AIバンドル）**

| 項目 | 内容 |
|------|------|
| ML予測 | 4本（売上・離脱・LTV・在庫 or 広告ROI）+ SHAP分析 |
| MTG | 月1回60分の施策MTG（人による解釈・実行支援） |
| レポート | 月次自動レポート（PDF） |
| バンドル | 商品登録支援 / SNS自動化アプリ 1本込み |
| 訴求 | ECコンサル人力 ¥30万比で「**半額＋ツール込み**」の物語 |

---

**L3: ¥250,000/月（戦略伴走、要パイロット検証）**

| 項目 | 内容 |
|------|------|
| 機能 | L2全機能 + 月2MTG + カスタムMLモデル + 四半期戦略レビュー |
| 前提条件 | **パイロット1社 × 6週間以上の実運用検証完了をもってのみ「L3 production-ready」と表記** |
| 現状ステータス | それまでは「L3プロトタイプ・パイロット検証前」 |

---

### 3.2 主要差別化（用語修正）

**撤退する用語:**

| 旧（v2） | 撤退理由 |
|----------|---------|
| ❌ 「逆SHAP What-Ifシミュレーター」 | バズワード化リスク。SHAPは2020年頃からの標準手法であり、DataRobot・H2O.ai等が同等機能を保有。差別化訴求として機能しない |

**採用する訴求:**

> ✅ **「ML予測 + 月次レビュー + IT補助金対応 + 日本特化（薬機法/景表法/季節性）」**

**主訴求3要素（全コミュニケーションで一貫使用）:**

1. **IT導入補助金対応**: 実質負担1/2、申請代行込み
2. **3ヶ月PoC階段**: ¥6万から始められる、効果実証後L2へ
3. **人+AI伴走**: ECコンサル半額、月1MTGで効果確認

---

### 3.3 MVPの定義

現状のMLリポジトリ（ec-business-alignment.md 準拠）における各アプリの位置づけ:

| アプリ | 層 | 現状 |
|--------|-----|------|
| ec-demo | L1基礎 | 完成 |
| ec-rfm | L1 | 完成 |
| ec-ad-roi | L1 | 完成 |
| ec-dashboard | L1完成ボトルネック | **ML + SHAP 追加が未完了 → L1 production-ready の必須要件** |
| ec-executive-dashboard | L2 | 完成 |
| ec-what-if | L2 | 完成 |
| ec-monthly-briefing | L2 | 完成 |

> ⚠️ **ec-dashboard への ML + SHAP 追加が「L1完成のボトルネック」。** L1受注前に必ず完了させること。

**成功基準:** デモ閲覧者の50%以上が「自社データで試したい」、診断セッション申込率10%以上

### 3.4 プロダクトロードマップ

| 時期 | 機能 | 目的 | 時給への影響 |
|------|------|------|------------|
| M1〜M3 | MVP（ec-dashboard ML化 + LP） | 仮説検証 | — |
| M4〜M6 | L1顧客データ取込・L2バンドル完成 | PMF確認 | L2単価の裏付け |
| M7〜M12 | **テンプレート化**（ECパック） | 運用効率化 | **1社あたり工数 2h → 1.3h に圧縮** |
| M13〜M18 | レポート自動生成、Next.js移行判断 | スケール準備 | **1社あたり工数 1.3h → 1h** |
| M19〜M24 | オンライン講座、ホワイトラベル | 収益多角化 | 稼働0hの収益源追加 |

> **テンプレ化目標を保守化**: v2の「2h→1h」は過楽観。実測補正を反映し「2h→1.3h（M7-M12）」→「1.3h→1h（M13-M18）」の2段階に修正。

## 4. チャネル戦略

### 4.1 仮説検証フェーズ（M1〜M3）

目的: 需要の存在を最小コストで検証。EC一本化集中期。

| チャネル | アクション | 期待効果 | コスト |
|---------|-----------|---------|--------|
| **ビザスク等 Expert Network** | M3登録、月2-3件単発相談（インタビュー＋見込み顧客発掘） | 月8-10商談の起点 | ¥0 |
| **知人紹介** | MBA・前職人脈から中堅EC関係者を紹介 | 月3-5商談 | ¥0 |
| **ECzine寄稿** | 月1本（事例ベース、具体的Tipsで差別化） | 認知拡大、問合せ導線 | ¥0 |
| **note / X** | 週3投稿（具体的Tips、補助金情報） | フォロワー300、リード獲得 | ¥0 |
| **IT補助金パートナー登録** | 認定ITコーディネーターと連携 | 紹介リード | ¥10,000 |

> ⚠️ **v2.1撤回: ウェビナー15名集客（M1-M3）**
> 無料SNSでの集客は平均5-8名（当日参加3-5名）であり、ファネル算数が成立しない。M1-M3はウェビナーを実施しない。共催ウェビナーはM4以降（§4.2参照）。

---

### 4.2 事業検証フェーズ（M4〜M6）

目的: PMF初期シグナル確認、有料顧客獲得。

| チャネル | アクション | 期待効果 |
|---------|-----------|---------|
| **共催ウェビナー** | 物流SaaS・決済SaaSと共同開催（集客リスク分散） | 参加20-30名→商談5件 |
| **60分無料診断セッション** | 商談→デモ→課題整理→L1提案 | 診断→契約CVR 30% |
| **事例記事** | L1パイロット顧客の成果を匿名事例化（ECzine掲載） | 信頼性担保 + SEO |
| **補助金パートナー紹介** | IT補助金申請支援込みでL1提案 | 紹介リード継続 |

---

### 4.3 本格プロモーション（M7〜M24）

目的: 安定獲得 + 紹介チャネル確立。

| チャネル | アクション | 期待効果 |
|---------|-----------|---------|
| **紹介プログラム** | 紹介者に初月無料特典 | 紹介経由30%（M24目標） |
| **コンテンツSEO** | ブログ累計50本 + EC事例5本 | PV 3,500/月 |
| **パートナー提携** | EC支援会社 / 物流SaaS 1社と提携 | パートナー経由案件 |
| **オンライン講座** | 有料講座（M15〜） | 収益多角化（稼働0h収益） |
| **Expert Network拡張** | GLG / AlphaSights 登録（M21〜） | スポット¥3〜15万/回 |

### 4.4 ファネル現実化

> 確定要素 §6 に基づき、v2ファネル算数を現実化。

**M1-M6 EC獲得ファネル（月次）:**

| 段階 | 数値 | チャネル |
|------|------|---------|
| 認知 | 月50リーチ | ECzine寄稿 + note記事 + X投稿 |
| 商談 | **月8-10商談** | ビザスク（Expert）+ 知人紹介 + 補助金パートナー紹介 |
| 診断 | 月3-4件（CVR 30%） | 60分無料診断セッション |
| 契約 | **月1社**（CVR 30%） | L1 ¥6万 PoC |

**v2からの撤回事項:**

| 撤回項目 | 理由 |
|---------|------|
| ❌ ウェビナー15名集客（M1-M3） | 無料SNS集客の実態は5-8名（当日参加3-5名）でファネル不成立 |
| ❌ 累計ファネル 30%×33%=10% 二段モデル | B2B SaaS実態は0.6-1.5%。過楽観モデルを撤回 |
| ❌ Shopify Community・EC-CUBE勉強会（M1-M3優先） | 属性ミスマッチ（購買権限者へのリーチが低い） |

### 4.5 競合比較表

> 確定要素 §10 に基づく競合整理。

| 競合 | 価格 | 脅威度 | 当社の戦略 |
|------|------|--------|---------|
| **Shopify Magic / Sidekick** | 無料 | **高** | 補完ポジション。Shopifyアプリ化も検討 |
| **Klaviyo** | $150〜 | **高** | CRM外（横断分析）に特化して差別化 |
| **GPT-5/Claude + Code Interpreter** | ¥3,000/月 | **高** | 「人による伴走」を中核化、純SaaS路線をやめる |
| **国産ECコンサル** | ¥30〜100万/月 | 中 | 「半額＋ツール込み」で価格訴求 |
| **DataRobot / H2O.ai** | ¥20〜100万/月 | 低 | 価格帯が異なり競合しない |

**空白地帯（自社定義、phase2と統一）:**
> ¥60,000〜¥250,000帯の「人+AI + 日本特化（薬機法/景表法/季節性）+ IT補助金対応」
