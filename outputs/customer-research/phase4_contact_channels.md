# Phase 4: 顧客獲得チャネル設計書

## 目的

EC事業者（Shopify利用、月商500〜1,500万円）3〜5名に対し、30分程度の課題ヒアリングインタビューを実施するための具体的なチャネル・操作手順を設計する。

## 前提条件

- EC業界に知人ゼロの状態からスタート
- 個人フリーランス（1人）
- 予算：最小限（月数千円〜1万円程度）
- 目的：売り込みではなく「課題ヒアリング」（30分インタビュー）
- リモート前提

### 1a. Shopifyコミュニティ / フォーラム

#### コミュニティの場所

| 名称 | URL | 特徴 |
|------|-----|------|
| Shopify Community Japan（メイン） | https://community.shopify.com/c/shopify-community-japan-jp/ct-p/jp | 公式日本語フォーラム。マーチャント・パートナーが参加 |
| Shopify ディスカッション | https://community.shopify.com/c/shopify-%E3%83%87%E3%82%A3%E3%82%B9%E3%82%AB%E3%83%83%E3%82%B7%E3%83%A7%E3%83%B3/ct-p/sd-jp | テーマ別ディスカッションボード |
| Shopify日本語ボード一覧 | https://community.shopify.com/c/jp/13 | カテゴリ別の掲示板 |

#### 具体的な操作手順

**Step 1: アカウント作成（5分）**
1. https://community.shopify.com にアクセス
2. 右上の「ログイン」をクリック
3. Shopifyアカウントがなければ「アカウントを作成」で新規登録（メールアドレスのみで可）
4. プロフィールを設定：名前・自己紹介に「フリーランスでEC支援を検討中。EC事業者の課題を理解したい」旨を記載

**Step 2: まず「読む」フェーズ（1〜2日）**
1. 「Shopify ディスカッション」ボードを開く
2. 「最新の投稿」でソートし、直近1ヶ月の投稿を30件ほど読む
3. よく出てくる悩み・課題のパターンをメモする（例：決済設定、集客、在庫管理、アプリ選定など）
4. 月商規模の大きそうな投稿者（質問内容から推定）をリストアップ

**Step 3: 価値提供する投稿（3〜5日目）**
1. 自分の専門領域に近い質問スレッドを見つける
2. 丁寧に回答を書く（単なる「ありがとう」ではなく具体的なアドバイス）
3. 以下のような独立した投稿も有効：

```
【タイトル】EC事業者の皆さまに質問：日々の運営で最も時間がかかっている業務は？

はじめまして。フリーランスでEC支援サービスの立ち上げを検討しています。
実際にShopifyで運営されている方の「リアルな声」をお聞きしたく投稿しました。

・日々の運営で最も手間がかかっている作業は何ですか？
・「これが自動化できたら嬉しい」と思う業務はありますか？
・外部に相談したいが、誰に聞けばいいかわからない課題はありますか？

営業目的ではなく、純粋にEC事業者の課題を理解したいという動機です。
お気軽にコメントいただけると嬉しいです。
```

**Step 4: 個別コンタクト**
1. コメントをくれた方、または活発に投稿している方のプロフィールを確認
2. コミュニティ内のDM機能（「メッセージを送る」）は限定的なため、プロフィールにSNSリンクがあればそこ経由でコンタクト
3. スレッド内で「もしよろしければ、もう少し詳しくお話を伺えませんか？」と返信し、連絡手段（メール・X等）を提示

#### 注意点
- Shopifyコミュニティは直接的な営業行為を禁止している。あくまで「情報交換」のスタンスを保つこと
- 最初から「インタビューさせてください」と書くのではなく、まず信頼関係を構築する

## 2. 低コストチャネル（月1万円以内）

### 2a. ビザスク（visasq）

#### サービス概要
ビザスクは、業界経験者（アドバイザー/エキスパート）に1時間単位でスポットコンサルができるマッチングプラットフォーム。登録アドバイザー数は国内最大級の60万人超。

#### 料金体系

| プラン | 1回あたり費用 | 特徴 |
|--------|-------------|------|
| ビザスクlite（セルフマッチング） | 15,000〜30,000円/1時間 | 自分でアドバイザーを検索・指名 |
| ビザスク interview（フルサポート） | 約100,000円/1時間 | 専任スタッフがマッチング |

**今回の用途では「ビザスクlite」を使用。1件あたり15,000〜20,000円が目安。**

#### 登録方法（10分）
1. https://visasq.co.jp にアクセス
2. 「無料で登録する」をクリック
3. メールアドレス、またはFacebook/LinkedInアカウントで登録
4. 氏名・所属・依頼目的を入力
5. 依頼者として登録完了（登録自体は無料）

#### EC事業者にインタビュー依頼する手順

**方法1: 指名相談**
1. ログイン後、「エキスパートを探す」ページへ
2. キーワード: `Shopify EC運営 月商` `ECサイト 運営責任者`
3. 検索結果からプロフィールを確認
4. 適切なエキスパートに「相談する」ボタンからメッセージを送信
5. 謝礼金額・日時・実施方法（Web会議）を相談

**方法2: 公募**
1. 「案件を公募する」から新規案件を作成
2. タイトル例：「Shopify利用EC事業者の方に運営課題についてヒアリング（30分）」
3. 依頼内容に詳細を記載
4. 謝礼金額を設定（15,000円程度）
5. エキスパートからの応募を待つ

#### メリット / デメリット

| メリット | デメリット |
|---------|----------|
| 確実にインタビューが実施できる | 1件15,000〜30,000円とコストが高い |
| 相手の経歴・専門が事前にわかる | 月商規模の指定が難しい場合がある |
| プラットフォームの信頼性が高い | 3〜5名で45,000〜100,000円の予算が必要 |
| 日程調整がスムーズ | 予算「月数千円」には収まらない |

#### 予算が限られる場合の対策
- 謝礼を最低ライン（15,000円）に設定
- 30分のインタビューとして半額交渉を試みる（応じてもらえない場合もある）
- **1〜2名はビザスク、残りは無料チャネルで確保**するハイブリッド戦略がおすすめ
