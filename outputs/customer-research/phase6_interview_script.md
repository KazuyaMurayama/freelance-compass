# Phase 6: 顧客インタビュースクリプト

> 作成日: 2026-03-18
> 対象: Shopify EC事業者（月商500〜1,500万円、アパレル/コスメ/雑貨）
> 所要時間: 30分
> 形式: オンライン（Zoom/Google Meet）

---

## インタビューの目的

Phase 1〜5のリサーチで構築した仮説を、実際のEC事業者との対話で検証する。
特に以下の6つの仮説を優先的に検証する：

| 仮説ID | 内容 | 検証の核心 |
|--------|------|-----------|
| H1 | 商品ページ改善に¥30,000の支払意欲 | 価格感度 |
| H2 | CRM未導入は「難しそう」が理由 | 行動障壁の特定 |
| H3 | 無料AI診断→有料サービスへの興味 | ファネル成立性 |
| H4 | 月額¥80,000は許容範囲 | 価格帯の妥当性 |
| H5 | 「AI活用パートナー」呼称の受容性 | ポジショニング |
| H6 | 最も改善したい指標の特定 | 需要の優先順位 |

---

## インタビュー前の準備

### 事前情報収集（インタビュー前に15分で実施）
1. 対象者のShopifyストアを確認（公開情報のみ）
2. 商品数、カテゴリ、価格帯をメモ
3. SNSアカウントの投稿頻度・フォロワー数を確認
4. Googleで「ストア名 + レビュー」を検索

### 用意するもの
- 本スクリプト（印刷 or 画面表示）
- メモ用のスプレッドシート（後述のテンプレート）
- 録音の許可取得（冒頭で確認）

## スクリプト本体

### 0. アイスブレイク（2分）

```
本日はお忙しい中お時間いただきありがとうございます。
[名前]と申します。EC事業者の方々が日常的にどんな課題を
感じていらっしゃるか、率直なお話を聞かせていただければと思っています。

売り込みは一切ありません。純粋にお話を聞かせていただく場です。
もし差し支えなければ、メモのために録音させていただいてもよろしいですか？

最初に、かんたんに自己紹介をお願いできますか？
どんな商品を扱っていらっしゃるか、ざっくりで大丈夫です。
```

**聞き取りポイント**: 商品カテゴリ、月商規模（直接聞かない、自然に出てくるのを待つ）、運営年数、スタッフ数（1人か複数か）

---

### 1. 日常業務の実態（8分）

**目的**: 時間の使い方のリアルな実態を把握する

#### Q1-1: 典型的な1日の流れ
```
ECストアの運営で、典型的な1日の流れを教えていただけますか？
朝起きてから、どんな順番でどんな作業をされていますか？
```

**深掘りポイント**:
- 「それにはだいたいどのくらい時間がかかりますか？」
- 「その作業は毎日やるものですか？週1くらいですか？」
- 「その中で一番"これやりたくないな"と思う作業はどれですか？」

#### Q1-2: 最も時間がかかっている作業
```
EC運営の中で、「もっと楽にならないかな」と一番感じる作業は何ですか？
```

**深掘りポイント**:
- 「具体的にどの部分が大変ですか？」（例: 商品登録なら撮影？テキスト？登録操作？）
- 「それを解決するために、これまで何か試したことはありますか？」
- 「それはうまくいきましたか？」

#### Q1-3: 想定外の発見を探る質問
```
EC運営を始める前と比べて、「思っていたより大変だった」ことはありますか？
```

---

### 2. お金の使い方（7分）

**目的**: 支払意欲と価格感度を探る（H1, H4の検証）

#### Q2-1: 現在の外部サービス利用状況
```
今、EC運営のために月々いくらくらいツールやサービスに払っていますか？
Shopifyの月額以外で、使っているツールやサービスがあれば教えてください。
```

**深掘りポイント**:
- 各ツール/サービスの名前、月額、満足度
- 「やめようと思ったツールはありますか？なぜですか？」
- 「逆に、"これには助けられている"というツールはありますか？」

#### Q2-2: 外注の経験
```
作業の一部を外注したことはありますか？
例えば、商品撮影、ページ制作、広告運用、コンサルなど。
```

**深掘りポイント（外注経験ありの場合）**:
- 「いくらくらいでしたか？」
- 「期待通りの結果でしたか？」
- 「次も同じサービスを使いますか？なぜ？」

**深掘りポイント（外注経験なしの場合）**:
- 「外注を検討したことはありますか？」
- 「外注しない理由は何ですか？」（コスト？信頼？自分でやりたい？）

#### Q2-3: 価格感度（H1, H4）
```
仮に、AIを使ってストアの商品ページを20〜30点まとめて改善してくれるサービスが
あったとして、いくらくらいなら試してみようと思いますか？

逆に、「この金額以上は絶対払わない」というラインはどのあたりですか？
```

**Van Westendorp法の4質問**（時間があれば）:
1. 「いくらから"安すぎて不安"だと感じますか？」
2. 「いくらから"安くてお得"だと感じますか？」
3. 「いくらから"高いけど検討する"と感じますか？」
4. 「いくらから"高すぎて無理"と感じますか？」

---

### 3. データ活用とCRM（5分）

**目的**: CRM/データ分析の実態と障壁を把握（H2, H6の検証）

#### Q3-1: データの見方
```
売上のデータは普段どうやって見ていますか？
Shopifyの管理画面？GA4？それとも特に見ていない？
```

**深掘りポイント**:
- 「そのデータを見て、何かアクションを変えたことはありますか？」
- 「"この数字が分かればいいのに"と思うことはありますか？」

#### Q3-2: リピーター施策（H2）
```
リピーターのお客様を増やすために、何か施策はされていますか？
例えば、メルマガ、LINE、ポイント制度など。
```

**深掘りポイント（やっている場合）**:
- 「どのくらいの頻度で配信していますか？」
- 「効果は感じていますか？」
- 「一番大変なのはどの部分ですか？」

**深掘りポイント（やっていない場合）**:
- 「やった方がいいとは思いますか？」
- 「やらない理由は何ですか？」→ **ここがH2の核心**
  - a) 時間がない
  - b) やり方が分からない
  - c) ツールの使い方が難しい
  - d) 効果が分からないから優先度が低い
  - e) そもそも必要性を感じない

#### Q3-3: 最も改善したい指標（H6）
```
もし1つだけ数字を改善できるとしたら、何を改善したいですか？

1. 新規のお客様の数
2. 1回あたりの注文金額
3. リピート率
4. 商品ページの購入率（CVR）
5. その他
```

---

### 4. AI活用と将来の展望（5分）

**目的**: AI活用の認知度と期待値を把握（H3, H5の検証）

#### Q4-1: AI利用状況
```
ChatGPTやAIツールをEC運営で使ったことはありますか？
```

**深掘りポイント（使っている場合）**:
- 「具体的にどんな場面で使っていますか？」
- 「うまくいった使い方と、期待はずれだった使い方はありますか？」

**深掘りポイント（使っていない場合）**:
- 「使ってみたいとは思いますか？」
- 「何に使えたら嬉しいですか？」

#### Q4-2: AI診断への反応（H3）
```
もし無料で、AIがあなたのShopifyストアを自動分析して、
「ここを改善すれば売上が上がる可能性が高い」というレポートを
出してくれるサービスがあったら、使ってみたいと思いますか？
```

**深掘りポイント**:
- 「どんな情報が含まれていたら"使ってみたい"と思いますか？」
- 「逆に、"怪しいな"と思うポイントはありますか？」
- 「ストアのデータを連携することに抵抗はありますか？」

#### Q4-3: サービス呼称への反応（H5）
```
こういったサービスを提供する人がいるとして、
「ECコンサルタント」と「AI活用パートナー」では、
どちらの方が相談しやすいと感じますか？
```

---

### 5. クロージング（3分）

```
非常に参考になるお話をありがとうございました。
最後に2つだけ伺わせてください。

1. 今のEC運営で一番の悩みを1つだけ挙げるとしたら、何ですか？

2. もし今後、実際にAIを活用したストア改善のサービスを
   始めることになったら、ご連絡してもよろしいですか？
   もちろん、その時に改めてご判断いただければ大丈夫です。
```

**重要**: 2の質問で「はい」と答えた人は、将来の初期クライアント候補。連絡先を確実に取得する。

## インタビュー分析フレームワーク

5名のインタビュー完了後、以下の分析を行う：

### 1. 仮説別の集計

| 仮説 | 支持 | 否定 | 保留 | 判定 |
|------|------|------|------|------|
| H1 | /5 | /5 | /5 | Go/NoGo/要追加検証 |
| H2 | /5 | /5 | /5 | |
| H3 | /5 | /5 | /5 | |
| H4 | /5 | /5 | /5 | |
| H5 | /5 | /5 | /5 | |
| H6 | /5 | /5 | /5 | |

**Go基準**: 5名中3名以上が「支持」→ 仮説は有効
**NoGo基準**: 5名中3名以上が「否定」→ 仮説を棄却、ピボット検討
**要追加検証**: 上記以外 → 追加5名のインタビューを検討

### 2. 共通パターンの抽出

```
- 全員に共通する課題:
- 全員に共通する行動パターン:
- 予想外だった発見（2名以上に共通）:
- 価格帯の合意水準:
```

### 3. ソリューション案の修正判断

| インタビュー結果 | Phase 5のソリューション案への影響 | 修正内容 |
|----------------|-------------------------------|---------|
| H1否定（¥30,000高い） | Step 2の価格調整 | ¥15,000〜20,000に下方修正 |
| H2否定（CRM不要） | Step 2施策Bの優先度下げ | 施策A（商品ページ）に集中 |
| H3否定（無料診断に不信感） | Step 1の設計変更 | 「診断」→「無料レポート」に呼称変更 |
| H4否定（月額¥80,000高い） | Step 3の価格設計見直し | ライトプラン¥30,000からスタート |
| H6の結果 | 最も需要のある指標にフォーカス | ダッシュボードのKPI変更 |
